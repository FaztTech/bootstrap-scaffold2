# Documentation
- [harp](http://harpjs.com)

# Create a Project
you can use the generator command or use this structure.
- npm i harp -g
- harp init myproject
- harp server myproject

# Usefull Commands
- compile assets with harp: `harp compile`
- server assets with harp: `harp server`
